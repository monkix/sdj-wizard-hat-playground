# SdJ x Wizard Hat Playground

Interactive learning playground for exploring Spiel des Jahres winners through WoL's Wizard Hat and MAGIC Star lenses.

## Current MVP

- Award Game Explorer timeline for 2015-2025 winners across Spiel, Kennerspiel, and Kinderspiel.
- Mechanic filters for Wizard Hat CORE/TASTE patterns.
- Game detail panel with MAGIC Star, Wizard Hat mapping, table moment, replay engine, and design lesson.
- Future-ready rails for Wizard Hat Lab and Award Pattern Challenge.

## Run

```powershell
npm install
npm run dev
```

## Deploy

This app is configured for GitHub Pages. Push to `main` and the `Deploy to GitHub Pages` workflow will build and publish the Vite app.
